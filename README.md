# Question and Answer:-
**1.  What is the difference between var, let, and const?** <br>
**Ans:** var, let, const are uses in variable declaration. The main differences of var, let, const is var can be re-declared but let and const can't be re-declared.
var and let can be updated but const can't be updated. var is function scoped but let and const are block scope. <br> <br>
**2.  What is the spread operator (...)?** <br>
**Ans:** The spread operator is a operator that expand or spread elements of an array or object into individual elements. <br><br>
**3. What is the difference between map(), filter(), and forEach()?** <br>
**Ans:** The main differences betwenn map(), filter(), and forEach() is:- <br><br> map() method is used when we want to modify an array or object, filter() is used when 
we need a specific element from an array or object and forEach()  is used when we just loop through inside the array or object. map() will return a new array, filter() will
return a new array but forEach() method doesn't return any thing.<br><br>
